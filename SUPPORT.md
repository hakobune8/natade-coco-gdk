# Support

This kit supports the documented Game Schema, Protocol, Controller SDK,
Display SDK, build, validation, and deployment-handoff contracts.

Before opening an issue, run `make validate`, search existing issues, and reduce
the problem to the generated starter when possible. Provide versions, browser
and OS, the failing command, and sanitized logs. Never post credentials,
tokens, reconnect handles, private DNS names, player data, or IP addresses.

The project does not provide an SLA. k3s, Fleet, DNS, TLS, Wi-Fi, kiosk, Edge
hardware, Registry operation, game-specific content, or production deployment
approval are owned by their respective operators or game repositories.
